# NarrativeOS

> The Cinematic AI-Native Operating System for IBM Bob Repository Intelligence

NarrativeOS transforms repositories into living, explorable software systems through cinematic architecture visualization, AI-guided walkthroughs, governance intelligence, and predictive impact analysis powered by IBM Bob.

---

# Overview

Modern repositories are:
- deeply interconnected
- difficult to onboard into
- architecturally invisible
- risky to modify

Existing developer tooling exposes:
- raw code
- static dependency graphs
- disconnected dashboards

But developers still struggle to answer:

> “How does this system actually work?”

NarrativeOS solves this by turning repositories into interactive intelligence systems.

Developers can paste a GitHub repository link and instantly explore:
- architecture topology
- execution flow
- dependency propagation
- governance risks
- impact simulations
- AI-guided walkthroughs

through a cinematic AI-native interface powered by IBM Bob.

# Getting Started

## Installation

```bash
git clone <repository-url>
cd narrativeos
npm install
```

---

## Environment Variables

Create a `.env.local` file:

```env
GITHUB_TOKEN=
IBM_BOB_API_KEY=
```

Both are optional.

NarrativeOS includes graceful fallback intelligence when API keys are unavailable.

---

## Run Development Server

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
